# Backend API

NestJS API for resilient checkout flow.

## Docker Compose

Use root-level compose from repository root:

```bash
docker compose up --build
```

## Run

```bash
npm install
npm run start:dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm run test
npm run test:e2e
npm run test:cov
```

## Docs

- Swagger: `http://localhost:3002/api/docs`
